Precompiled Java package

The precompiled jar contains the complete Arabesque system along with all its dependencies. It also includes 4 example applications: clique finding, motif counting, frequent subgraph mining and triangle counting.

This jar is ideal if you simply want to test the example applications or manually setup your own project.

Download the latest JAR

To run any of the example applications, refer to our user guide.

Preconfigured Maven project

If you want to start developing your own graph mining implementations on top of Arabesque, the easiest way to achieve this is by forking our Arabesque-Skeleton project and follow the instructions on the README file.
